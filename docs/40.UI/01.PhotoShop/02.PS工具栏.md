## 移动工具：V

PS 自动勾选了属性栏中的 `自动选择`，这样用户就可以随意选择不同图层的图形。如果取消勾选了该选项，则你只能拖动当前图层的图形，当然你也可以使用 <kbd>Ctrl</kbd> + <kbd>鼠标左键</kbd> 拖动其他图层的图形

## 选区工具

- 矩形选框工具：矩形选框工具、椭圆选框工具、单行选框工具、单列选框工具
- 套索工具：套索工具、多边形套索工具、磁性套索工具
- 对象选择工具：对象选择工具、快速选择工具、魔棒工具

如果要在 Photoshop 中处理图像的局部效果，就需要为图像指定一个有效的编辑区域，这个区域就是 PS 选区。通过选择特定区域，可以对该区域进行编辑并保持未选定区域不被改动。另外，使用选区可以将对象从一张图像中分离出来。注意，将选区中的图像分离出来，必须选择『移动工具』

![属性栏](https://cdn.jsdelivr.net/gh/sbgong/static/blog/ui/ps/202207121222038.png)

- 新选区：默认选中；激活该按钮以后，可以创建一个新选区。如果已经存在选区，那么新创建的选区将替代原来的选区
- 添加到选区：激活该按钮以后，可以将当前创建的选区添加到原来的选区中（按住 <kbd>Shift</kbd> 键也可以实现相同的操作）
- 从选区减去：激活该按钮以后，可以将当前创建的选区从原来的选区中减去（按住 <kbd>Alt</kbd> 键也可以实现相同的操作）
- 与选区交叉：激活该按钮以后，新建选区时只保留原有的选区与新创建的选区相交的部分（按住 <kbd>Alt</kbd> + <kbd>Shift</kbd>键也可以实现相同的操作）
- 反选：创建一个选区之后，按住 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>i</kbd> 键，即可反选该选区之外的内容
- 羽化选区：羽化原理是令选区内外衔接部分虚化，起到渐变的作用从而达到自然衔接的效果
  - 使用方法一：先调整好羽化数值，然后再新建选区
  - 使用方法而：已经新建好选区之后还想羽化，就需要使用快捷键 <kbd>Shift</kbd> + <kbd>F6</kbd> 调出羽化参数窗口进行设置
- 取消选区快捷键：<kbd>Ctrl</kbd> + <kbd>Delete</kbd> 键；重新调出选区快捷键：<kbd>Ctrl</kbd> + <kbd>鼠标左键</kbd> 点击想要的选区图层即可

## 污点修复画笔工具

PS 修复工具用于对图像细微部分进行修整，是在处理图像时不可缺少的工具之一

包括：污点修复画笔工具、修复画笔工具、修补工具、内容感知移动工具、红眼工具

- 污点修复画笔工具：使用污点修复画笔工具可以快速消除图像中的污点和某个对象。它不需要设置取样点，因为它可以自动从所修饰区域的周围进行取样。调整修补范围的快捷键：<kbd>Alt</kbd> + <kbd>鼠标右键</kbd> + <kbd>鼠标左右移动</kbd> 来缩小和扩大范围（左小右大）
- 修复画笔工具：按住 <kbd>Alt</kbd> 键，然后选择好的地方，进行选取样本，然后松开 <kbd>Alt</kbd> 键，点击污点地方，就能将污点进行消除，有点类似图章的效果。还能调整选择的硬度：<kbd>Alt</kbd> + <kbd>鼠标右键</kbd> + <kbd>鼠标上下移动</kbd>（上软下硬）
- 修补工具：你自己手动选择污点的区域，然后把它拖到好的地方去
- 内容感知移动工具：圈选图像部分，把它移动到指定的位置（相当于剪切），并且它会自动内容感知指定位置的周围，让这两部分尽可能融合在一起。建议：刚开始选择图像的时候，尽量较小圈选图像的周围，这样有利于后面的融合；还可以把结构属性调成最小、颜色属性调为最大，也更有利于融合
- 红眼工具：就是去除眼睛的红色

## 仿制图章工具

使用 PS 仿制图章工具，可以将图像的一部分复制到同一图像的另一个位置上，或绘制到具有相同颜色模式的任何打开的文档的另一部分，当然也可以将一个图层的一部分绘制到另一个图层上

它包括：仿制图章工具、图案图章工具

- 仿制图章工具应用前，需要按住<kbd>Alt</kbd>键，定义取样点
- 选定仿制源后，松开<kbd>Alt</kbd>键。把光标移到你所要修补的地方，重复点击鼠标，直到你所希望的效果出现
- 注意：默认放置图章工具勾选了『对齐』属性，它能按原图比例将整图复制出来

## 橡皮擦工具

PS 橡皮擦工具：图像擦除工具主要用了擦除多余的图像

- 橡皮擦工具：将像素更改为背景色或透明。使用橡皮擦工具在背景图层或锁定了透明像素的图层中进行擦除，则擦除的像素将变成背景色；如果在普通图层中进行擦除，则擦除的像素将变成透明
- 背景橡皮擦工具：在抹除背景的同时保留前景对象的边缘。背景橡皮擦工具是一种智能化的橡皮擦。设置好背景色后，使用该工具可以在抹除背景的同时保留前景对象的边缘（调小容差）
- 魔术橡皮擦工具：将所有相似的像素更改为透明。使用该工具在图像中单击时，可以将所有相似的像素更改为透明，效果等同于魔棒工具

ps 橡皮擦工具放大缩小快捷键：<kbd>]</kbd> 键和<kbd>[</kbd>键（可进行微调）

## 前景色和背景色

- 填充前景色：<kbd>Alt</kbd> + <kbd>Delete</kbd>
- 填充背景色：<kbd>Ctrl</kbd> + <kbd>Delete</kbd>
- 填充：在当前图层或选区内填充颜色或图案或内容识别：<kbd>Shift</kbd> + <kbd>F5</kbd>
- 描边：在选区、路径或图层周围创建任何颜色的边框：<kbd>Alt</kbd> + <kbd>E</kbd> + <kbd>S</kbd>
