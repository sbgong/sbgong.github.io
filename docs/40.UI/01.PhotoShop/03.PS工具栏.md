|       工具       | 快捷键 | 说明                                 |
| :--------------: | :----: | ------------------------------------ |
|     移动工具     |  `v`   | 移动选区或图层                       |
|   矩形选框工具   |  `m`   | 创建矩形形状的选区                   |
|     套索工具     |  `l`   | 创建手绘选区                         |
|   对象选择工具   |  `w`   | 查找并自动选择对象                   |
|     裁剪工具     |  `c`   | 裁切或扩展图像的边缘                 |
|     图框工具     |  `k`   | 为图像创建占位符图框                 |
|     吸管工具     |  `i`   | 从图像中取样颜色                     |
| 污点修复画笔工具 |  `j`   | 移去标记和污点                       |
|     画笔工具     |  `b`   | 绘制自定义画笔描边                   |
|   仿制图章工具   |  `s`   | 使用来自图像其他部分的像素绘画       |
| 历史记录画笔工具 |  `y`   | 建图像的某些部分恢复到以前的状态     |
|    橡皮擦工具    |  `e`   | 将像素更改为背景颜色，或者使它们透明 |
|     渐变工具     |  `g`   | 创建颜色之间的渐变混合               |
|     模糊工具     |        | 模糊图像中的区域                     |
|     减淡工具     |  `o`   | 调亮图像中的区域                     |
|     钢笔工具     |  ` p`  | 通过锚点与手柄创建和更改路径或形状   |
|   横排文字工具   |  `t`   | 添加横排文字                         |
|   路径选择工具   |  `a`   | 选择整个路径                         |
|     矩形工具     |  `u`   | 绘制矩形                             |
|     抓手工具     |  `h`   | 在图像的不同部分间平移               |
|     缩放工具     |  `z`   | 在图像上放大和缩小                   |

## 移动工具：V

PS 自动勾选了属性栏中的 `自动选择`，这样用户就可以随意选择不同图层的图形。如果取消勾选了该选项，则你只能拖动当前图层的图形，当然你也可以使用 <kbd>Ctrl</kbd> + <kbd>鼠标左键</kbd> 拖动其他图层的图形
