## HTML 基础标签

|     标签元素      | 说明                                                                         |
| :---------------: | ---------------------------------------------------------------------------- |
| `<!--注释内容-->` | 注释标签                                                                     |
| `<!DOCTYPE html>` | 定义文档类型                                                                 |
|     `<html>`      | HTML 文档跟标签                                                              |
|     `<head>`      | 用于定义文档的头部，绝大多数文档头部包含的数据都不会真正作为内容显示给读者   |
|     `<body>`      | 定义文档的主体，包含文档的所有内容（比如文本、超链接、图像、表格和列表等等） |
|       `<p>`       | 段落标签；p 标签会自动在相邻的段落之间插入一些垂直的间距（块级元素）         |
|      `<hr>`       | 水平线 <Badge text="闭合标签" />                                             |
|      `<br>`       | 换行标签 <Badge text="闭合标签" />                                           |

## head 头标签

|  标签元素  | 说明                                                                                           |
| :--------: | ---------------------------------------------------------------------------------------------- |
| `<title>`  | 定义文档的标题，为 head 标签必备子标签                                                         |
|  `<base>`  | 为页面上的所有链接规定默认地址或默认目标（target）<Badge text="闭合标签" />                    |
|  `<meta>`  | 提供关于 HTML 文档的元数据，比如针对搜索引擎和更新频度的描述和关键词 <Badge text="闭合标签" /> |
|  `<link>`  | CSS：链接样式表，定义文档与外部资源的关系 <Badge text="闭合标签" />                            |
| `<style>`  | CSS：为 HTML 文档定义样式信息                                                                  |
| `<script>` | 用于定义客户端脚本，比如 JavaScript                                                            |

## 文本相关

> b i s u 没有强调的意思，就是标签本身的意思；strong em del ins 有语义，是起到加重语气的效果

|         标签元素         | 说明                                                          |
| :----------------------: | ------------------------------------------------------------- |
|       `<h1>-<h6>`        | 标题标签：1 最大/6 最小（块级元素）                           |
| `<font>` 和 `<basefont>` | 定义 HTML 字体 <Badge text="HTML5不支持" type="warning"/>     |
|        `<center>`        | 定义居中的内容 <Badge text="HTML5不支持" type="warning"/>     |
|          `<b>`           | 加粗                                                          |
|          `<i>`           | 斜体                                                          |
|          `<s>`           | 删除线                                                        |
|          `<u>`           | 下划线                                                        |
|        `<strong>`        | 加粗；加重语气                                                |
|          `<em>`          | 斜体；加重语气                                                |
|         `<del>`          | 删除线；加重语气                                              |
|         `<ins>`          | 下划线；加重语气                                              |
|         `<big>`          | 大号字                                                        |
|        `<small>`         | 小号字                                                        |
|         `<sub>`          | 下标                                                          |
|         `<sup>`          | 上标                                                          |
|        `<strike>`        | 删除线文本 <Badge text="废弃" type="warning"/>；使用 del 代替 |

## 超链接/图像

|  标签元素  | 说明                               |
| :--------: | ---------------------------------- |
|   `<a>`    | 超链接（行级元素）                 |
|  `<img>`   | 图像（行级元素）                   |
|  `<map>`   | 图像地图                           |
|  `<area>`  | 图像地图中的可点击区域             |
| `<canvas>` | 定义图形                           |
|  `<svg>`   | 定义 SVG 图形的容器                |
|  `<div>`   | 块级标签；浏览器会在其前后显示折行 |
|  `<span>`  | 行级标签                           |


## 无序/有序列表

| 标签元素 | 说明                                         |
| :------: | -------------------------------------------- |
|  `<ul>`  | 无序列表；使用粗体圆点进行标记（块级元素）   |
|  `<ol>`  | 有序列表；使用数字进行标记（块级元素）       |
|  `<li>`  | 列表项目；可用在有序和无序之间               |
|  `<dl>`  | 定义列表；用于结合`<dt>`和`<dd>`（块级元素） |
|  `<dt>`  | 定义列表中的项目                             |
|  `<dd>`  | 描述列表中的项目                             |

## 表格相关

|   标签元素   | 说明                 |
| :----------: | -------------------- |
|  `<table>`   | 表格（块级元素）     |
| `<caption>`  | 表格标题             |
|    `<th>`    | 表格表头             |
|    `<tr>`    | 表格行               |
|    `<td>`    | 表格单元（行级元素） |
|  `<thead>`   | 表格头部             |
|  `<tbody>`   | 表格主体             |
|  `<tfoot>`   | 表格底部             |
|   `<col>`    | 表格列的属性         |
| `<colgroup>` | 表格列的组           |

## 表单相关

|   标签元素   | 说明                                                                                    |
| :----------: | --------------------------------------------------------------------------------------- |
|   `<form>`   | 定义供用户输入的 HTML 表单                                                              |
|  `<input>`   | 用于搜集用户信息，根据不同的 type 属性有很多的形态 <Badge text="闭合标签" />            |
|  `<output>`  | 定义输出的一些类型                                                                      |
|  `<select>`  | 定义下拉列表；select 元素中的 option 标签用于定义列表中的可用选项                       |
|  `<option>`  | 定义下拉列表中的一个选项；默认会把首个选项显示为被选选项                                |
| `<optgroup>` | 定义选择列表中相关选项的组合                                                            |
| `<textarea>` | 定义多行输入字段（文本域）                                                              |
|  `<button>`  | 定义可点击的按钮                                                                        |
|  `<label>`   | 定义 input 元素的标记，当用户选择该标签时，浏览器会自动将焦点转到和标签相关的表单控件上 |
| `<datalist>` | 为 input 元素规定预定义选项列表；用户会在他们输入数据时看到预定义选项的下拉列表         |
| `<fieldset>` | 可将表单内的相关元素分组                                                                |
|  `<legend>`  | legend 元素为 fieldset 元素定义标题                                                     |
|  `<keygen>`  | 定义生成密钥                                                                            |

## 新增语义标签

|    标签元素    | 说明                                                                                                |
| :------------: | --------------------------------------------------------------------------------------------------- |
|   `<header>`   | 定义 section 或 page 的页眉；页眉通常包括网站标志、主导航、全站链接以及搜索框（块级元素）           |
|    `<nav>`     | 导航链接；仅对文档中重要的链接群使用（块级元素）                                                    |
|    `<main>`    | 文档主内容；一个页面只能使用一次（块级元素）                                                        |
|  `<article>`   | 文章；定义页面独立的内容，它可以有自己的header、footer、sections等（块级元素）                      |
|  `<section>`   | 章节；元素用于标记文档的各个部分，例如长表单文章的章节或主要部分（块级元素）                        |
|   `<aside>`    | 侧边栏（块级元素）                                                                                  |
|   `<footer>`   | 定义 section 或 page 的页脚；通常包含文档的作者，著作权信息，链接的使用条款，联系信息等（块级元素） |
|  `<details>`   | 定义用户能够查看或隐藏的额外细节                                                                    |
|  `<summary>`   | 定义 details 元素的可见标题                                                                         |
|   `<figure>`   | 规定自包含内容，比如图示、图表、照片、代码清单等（块级元素）                                        |
| `<figcaption>` | 定义 figure 元素的标题                                                                              |
|    `<mark>`    | 定义重要的或强调的文本                                                                              |
|    `<time>`    | 定义日期/时间                                                                                       |
|   `<dialog>`   | 定义对话框或窗口                                                                                    |
|   `<meter>`    | 定义已知范围或分数值内的标量测量；例如磁盘用量、查询结果                                            |
|  `<progress>`  | 定义任何类型的任务的进度条                                                                          |
|    `<wbr>`     | 规定在文本中的何处适合添加换行符                                                                    |

## 内联框架

|   标签元素   | 说明                                                                                        |
| :----------: | ------------------------------------------------------------------------------------------- |
| `<frameset>` | 定义框架集 <Badge text="HTML5不支持" type="warning"/>                                       |
|  `<frame>`   | 定义框架集的窗口或框架 <Badge text="HTML5不支持" type="warning"/> <Badge text="闭合标签" /> |
| `<noframes>` | 定义针对不支持框架的用户的替代内容 <Badge text="HTML5不支持" type="warning"/>               |
|  `<iframe>`  | 定义内联框架                                                                                |
| `<noscript>` | 定义在脚本未被执行时的替代内容（文本）                                                      |


## 计算机输出标签

| 标签元素 | 说明                                                                           |
| :------: | ------------------------------------------------------------------------------ |
| `<pre>`  | 预格式化文本；被包围在 pre 元素中的文本通常会保留空格和换行符                  |
| `<kbd>`  | 定义键盘文本。它表示文本是从键盘上键入的。它经常用在与计算机相关的文档或手册中 |
| `<code>` | 定义编程代码示例；不保留多余的空格和折行                                       |
| `<samp>` | 定义计算机输出示例；通常使用浏览器缺省的 monotype 字体                         |
|  `<tt>`  | 呈现类似打字机或者等宽的文本效果                                               |
| `<var>`  | 定义数学变量                                                                   |

## 引用和术语定义

|    标签元素    | 说明                                                                                         |
| :------------: | -------------------------------------------------------------------------------------------- |
|    `<abbr>`    | 定义缩写；可以使用 title 属性，这样就能够在鼠标指针移动到该元素上时显示出简称/缩写的完整版本 |
|  `<acronym>`   | 定义首字母缩写 <Badge text="HTML5不支持" type="warning"/>                                    |
|    `<dfn>`     | 定义术语元素；默认使用斜体来展现该标签内的文本                                               |
|  `<address>`   | 定义地址块                                                                                   |
|    `<bdo>`     | 定义文字方向                                                                                 |
| `<blockquote>` | 定义长的引用；通常会对该元素进行缩进处理，在元素前后添加了换行，并增加了外边距               |
|     `<q>`      | 定义短的引用；浏览器在引用的周围插入了引号                                                   |
|    `<cite>`    | 表示所包含的文本对某个参考文献的引用，比如书籍或者杂志的标题；默认以斜体显示                 |

## 媒体相关

|  标签元素  | 说明                                                |
| :--------: | --------------------------------------------------- |
| `<audio>`  | 定义音频                                            |
| `<video>`  | 定义视频                                            |
| `<source>` | 为媒介元素（比如 video 和 audio）定义媒介资源       |
| `<track>`  | 为诸如 video 元素之类的媒介规定外部文本轨道；即字幕 |
| `<embed>`  | 定义嵌入的内容，比如插件                            |
| `<object>` | 定义嵌入的对象                                      |
| `<param>`  | 定义对象的参数                                      |
