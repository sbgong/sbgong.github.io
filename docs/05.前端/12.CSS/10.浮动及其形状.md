在很长一段时间内，浮动元素是所有 Web 布局方案的基础（很大程度上依赖 clear 属性）。但是浮动并不是为布局而生的，与使用表格布局基本一样，把浮动作为布局工具算是无奈之举，那时别无他选。然而，浮动自身确实相当有趣和有用的，尤其是最近新增浮动形状之后，内容可以沿非矩形边框浮动了

## 浮动：float

CSS可以浮动任何元素，通过 `float` 属性实现。例如：图像浮动到右侧，其他内容（例如文本）将围绕图像流动。浮动主要用于盒子模型，可以让多个盒子一行展示，注意浮动元素不会压住标准流的文字

| 功能   | 说明                  |
| ------ | --------------------- |
| 取值   | `left / right / none` |
| 初始值 | none                  |
| 适用于 | 所有元素              |
| 继承性 | 否                    |
| 动画性 | 否                    |
